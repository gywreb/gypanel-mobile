# ADMIN-PANEL-MOBILE ⚙️ 

## Built with
- React Native
- Redux (Thunk, Persist)
- React Navigation
- VictoryJS (Chart)
- Axios (Api Client)

## Installing dependencies

https://reactnative.dev/docs/environment-setup

## Getting Started

#### 1. Install

```bash
# Dependencies Install
npm install
# or
yarn install

# Install Pods
cd ios && pod install
# or
npx pod-install
```

#### 2. Run

```
yarn android

```
or

```
npx react-native run-android
```

Replace android = ios for ios dev
